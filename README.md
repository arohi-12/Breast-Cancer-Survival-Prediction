# 🧠 Breast Cancer Survival Prediction

## 📌 Project Overview
This project aims to predict the survival of breast cancer patients using machine learning techniques. The model is trained on clinical and demographic data to classify whether a patient is likely to survive.

---

## 📊 Dataset
- The dataset contains patient medical records
- Includes features like age, tumor size, treatment, etc.
- Target variable: Survival (Yes/No)

---

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit (for deployment)
